# Payzoris

> Virtual cards, infinite possibilities.

Create secure virtual cards instantly for online shopping, subscriptions, and business expenses. Control spending, protect your real card details, and manage your finances from one sleek dashboard.

## Design

Inspired by Lamborghini's brand system — true black canvas, gold accents, zero border-radius, all-caps typography. See [DESIGN.md](DESIGN.md) for the full design system.

## Stack

- HTML5
- CSS3 (no frameworks)
- Vanilla JavaScript

## Usage

Open `index.html` in any browser.
